# Screen Shader Feature

Time-of-day screen color overlay for ambient lighting.

## Components

### `<sz-screen-shader>`
Applies a CSS filter overlay that changes with time of day.

**Actions listened:**
- `shader:set-mode` — payload: mode string (`'off'`, `'css'`, `'webgl'`)

## Usage
```html
<sz-screen-shader></sz-screen-shader>
```
